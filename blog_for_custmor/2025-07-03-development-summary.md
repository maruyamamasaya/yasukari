# サイト開発状況サマリー

ヤスカリ公式サイトのリニューアルに向けた進捗をまとめました。README と Manual for System の内容を整理したものです。

## リリース履歴

README では初期リリースの **v0.50** から最新の **v0.59** までの変更点をまとめています。トップページの UI/UX 改善やチャットボット追加、ログイン機能の導入、店舗ブログ・ヒーロースライダー実装などが盛り込まれています。

## 7月1日時点の進捗

`manual_for_system/2025-07-01-development-summary.md` では以下を報告しています。

- CSS スタイルガイドの新設
- 最近チェックした商品機能の実装
- アクセス集中時の一時制限機能追加
- サイトリニューアルのお知らせ記事を公開

## 7月2日時点の進捗

`manual_for_system/2025-07-02-development-summary.md` では、誤って削除されたトップページセクションの復元や、カスタマーブログへのハッシュタグ追加、BikeModelCarousel のデザイン刷新などを記録しました。

## 実装済み機能

Next.js + TypeScript を基盤に、Hero スライダー、ブログカルーセル、FAQ アコーディオン、チャットボットウィジェット、最近チェックした商品表示など多彩なコンテンツを実装済みです。管理者向けブログは Markdown から静的生成しています。

## 今後の予定

モバイルレイアウト調整やアカウント機能の拡充、404 ページのデザインブラッシュアップ、本番公開に向けたインフラ整備と SEO 強化を進めていく予定です。
