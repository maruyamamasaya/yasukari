# サイトデザイン統一のためのCSSスタイルガイドまとめ

先日公開したCSSスタイルガイドのポイントを簡単にまとめました。開発に携わる方はぜひご一読ください。

- ドキュメントは `styles/global.css` を中心に、サイト全体のスタイル管理とカスタマイズ方法を説明しています。
- ブランドカラーやフォントセットをルート変数として定義し、ページをまたいで一貫したデザインを適用します。
- `body` の余白リセットやフォントサイズ、見出し・段落の余白など、基本スタイルの指針を示しています。
- リンクは通常下線を表示せずホバー時のみアンダーラインを付けるなど、細かなユーザビリティを向上させています。
- `.btn-primary` や `.card` などの共通コンポーネントを定義し、ホバー時の色変化や角丸などを統一しています。
- Swiper やモバイルナビゲーションといった UI 部品向けのスタイルも盛り込み、スマホ操作を意識した設計です。
- `@keyframes fade-up` を使ったシンプルなフェードアニメーションを定義し、要素に `animate-fade` を付与して利用できます。
- Tailwind CSS のユーティリティと `global.css` のクラスを組み合わせ、サイト全体の統一感を保つ方針です。

以上がCSSスタイルガイドの概要です。今後の開発・カスタマイズにお役立てください。
