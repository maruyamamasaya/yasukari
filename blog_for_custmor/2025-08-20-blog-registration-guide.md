# ブログ投稿手順

このファイルでは、顧客様向けブログの登録方法を説明します。

1. `/blog_for_custmor` ディレクトリに Markdown ファイルを作成します。
2. 先頭に以下のフロントマターを記述します。
   ```
   ---
   title: "記事タイトル"
   date: "2025-08-20"
   tags: お知らせ, キャンペーン
   eyecatch: 画像URL
   ---
   ```
3. 本文には Markdown 記法を利用できます。
4. `eyecatch` には Unsplash などの画像URLを指定してください。

サンプルとして上部のアイキャッチ画像に無料写真を設定しています。
