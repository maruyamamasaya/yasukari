# 開発状況サマリ (2025/07/02)

7/1 時点のまとめからの差分を中心に、最新の進捗を整理します。

## 7/1 からの主な変更点

- Manual for System ブログにキーワード検索できる `PostSearch.tsx` を追加
- 存在しないページ向けの `pages/404.tsx` を作成
- ChatBot のリサイズ方向がおかしかった点を修正
- 人気モデル一覧に価格情報付きバイクを追加
- 50cc スクーターのサンプルモデルを `lib/bikes.ts` へ追加

## 現状の実装済み機能

- Next.js + TypeScript によるサイト構築
- トップページでは Hero スライダーやブログカルーセル、FAQ、利用ガイド、最近チェックした商品などを表示
- 予約から返却までの流れを 3 ステップで解説
- チャットボットはカテゴリ選択と自由入力を備え、ウィジェットとして全ページから起動可能
- Manual for System は Markdown 記事を静的生成し、カレンダー表示・検索・ディレクトリツリーで記事を探しやすく整理
- ログインフォームとアクセス制限機能を試験運用中

- 404 ページのデザイン改善
- 本番公開に向けた環境構築と SEO 最適化
