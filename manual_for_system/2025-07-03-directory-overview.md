# ディレクトリ構成まとめ (2025/07/03)

このリポジトリで利用している主なディレクトリの役割を簡潔に整理します。開発を始める前に構造を把握しておくと、ファイルの配置場所を迷わずに済みます。

## 主要ディレクトリ

- `components/` - React コンポーネントを配置
- `pages/` - Next.js のルーティングに対応するページ群
- `manual_for_system/` - システムエンジニア・管理者向けの更新ブログ
- `blog_for_custmor/` - 店舗ブログ記事の Markdown ファイル
- `public/` - 画像などの静的ファイルを格納
- `styles/` - グローバル CSS やスタイル関連ファイル
- `lib/` - 各種ユーティリティ関数
- `data/` - FAQ などの静的データ
- `docs/` - プロジェクトの概要や利用方法を説明するドキュメント

## その他のファイル

- `next.config.js` や `tsconfig.json` など、プロジェクト設定に関するファイルはルートディレクトリにまとめています。

以上が基本的なディレクトリ構成です。詳しいファイルの内容や用途は、それぞれの README やドキュメントも併せて参照してください。
