# 開発状況サマリ (2025/07/01)

6/30 時点のまとめからの差分を中心に、現状の進捗を整理します。

## 6/30 からの主な変更点

- CSS スタイルガイドを新設し、共通デザインの調整方針を明確化
- 最近チェックした商品機能を実装し `localStorage` に履歴を保存
- アクセス集中時の一時制限機能を追加し `/wait` ページへ誘導
- サイトリニューアルのお知らせ記事を公開し移行計画を告知
- チャットボットにカテゴリーへ戻るボタンを追加、縦長表示で操作性向上
- トップページの利用ガイドを 3 ステップで簡潔にまとめて表示
- FAQ をアコーディオン形式に改良しデザインをアップデート
- ストア一覧への Google Maps 埋め込みなど各ページを拡充

## 現状の実装済み機能

- Next.js + TypeScript を基盤とする構成は継続
- トップページは Hero スライダーやブログカルーセル、FAQ、利用ガイド、最近チェックした商品などを含むモダンな構成
- チャットボットはカテゴリ選択と自由入力が可能で UI を改善
- Manual for System には CSS スタイルガイドや DDoS 対策手順など複数の記事を追加
- ログイン／サインアップ周りはフォーム枠を設置して試験中
- アクセス監視と制限機能を導入し安定運用を図る

## 今後の予定

- モバイル UI の最適化とアカウント機能の本格実装
- 公開に向けたデプロイ設定と SEO 調整
- リニューアルに伴う旧コンテンツの整理
