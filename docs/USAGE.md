# yasukari ご利用方法

このドキュメントではレンタルバイクサービス **yasukari** の基本的な利用の流れを説明します。

1. **店舗を選ぶ**
   - 足立小台本店または三ノ輪店から出発店舗を選択します。
2. **ご予約**
   - 各車両ページのスケジュールを確認し、クレジットカードで予約します。
   - 変更やキャンセルはお問い合わせよりご連絡ください。
3. **ご来店**
   - 予約当日は10:00〜18:30の間に免許証とヘルメットをご持参の上ご来店ください。
4. **ご利用・返却**
   - 契約者ご本人のみが乗車・返却可能です。
   - 返却はガソリン満タンの状態でお願いします。

より詳しい案内はサイト内の[ご利用案内ページ](/guide)をご確認ください。
