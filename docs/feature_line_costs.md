# 機能別のソース行数と人日換算（テスト工程除き）

以下は **1人日あたり200行** をベースに、主要機能群ごとのソースコード行数と人日、単価を試算したものです。人件費は **1人日=¥10,000** として計算しています。該当ディレクトリ・ファイル群の単純な行数合計であり、テスト・レビュー・管理工数は含みません。表は「いまあるコード量から見た工数感」をざっくり把握するためのもので、追加の要件調整や再設計が入る場合は別途上振れします。

- 線形換算のため、短い行数でも0.1人日などをそのまま計算しています。
- 行数はコメント・空行を含むため、実装密度によって実工数は前後します。
- テストやレビュー、設計フェーズは別途上乗せしてください。

## ざっくり合計感

- 行数合計（上記の単純合算）: **12,851行**
- 人日換算: **64.26人日** （12,851 ÷ 200）
- 人件費目安: **¥642,600** （64.26 × ¥10,000）

合計にはテストや運用準備は含まれないため、これらの工程を考慮する場合は20〜30%程度の上乗せを想定すると現実的です。

## メモ
- 行数は `wc -l` による単純計測のため、コメントや空行も含みます。
- 実装重複や共通コードを機能別に重複カウントしていないものの、複数機能で共有されるユーティリティは「汎用ライブラリ・API補助」にまとめています。
- テスト・受け入れ・運用準備は別途見積もりが必要です。
