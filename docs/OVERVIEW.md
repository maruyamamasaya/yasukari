# プロジェクト概要

このリポジトリはバイクレンタルサービス **yasukari** の公式サイトを Next.js で実装したものです。開発過程で追加された機能やリリース履歴をまとめています。

## リリース履歴

- **v0.50**: 初期構築、トップページUI、BikeModelCarousel などを実装
- **v0.51**: チャットボットウィジェットやホーム画面刷新、商品一覧ページ追加
- **v0.52**: 更新ブログのページネーション、DirectoryTree 折りたたみ、ソース閲覧ページ追加
- **v0.53**: 簡易ログイン機能を導入
- **v0.54**: 新着情報・店舗ブログやヒーロースライダー、法定ページなどを多数追加
- **v0.55**: トップページに FAQ セクションを配置
- **v0.56**: CSS スタイルガイド、最近チェックした商品機能、アクセス制限を追加
- **v0.57**: 人気モデルカルーセルをモダンなデザインへ更新
- **v0.58**: 店舗ブログにカレンダー・タグ別ページを追加し、ホームセクションも整理
- **v0.59**: 注目キーワードを7件まで表示し、最後にもっと見るリンクを追加

## 主要ディレクトリ

- `components/` - 各種 React コンポーネント
- `pages/` - Next.js のページ群
- `manual_for_system/` - 更新ブログや設計メモを置くドキュメント
- `blog_for_custmor/` - 店舗向けブログ記事の Markdown
- `data/` - FAQ などの静的データ
- `lib/` - 補助スクリプトやユーティリティ
- `styles/` - グローバル CSS
- `docs/` - プロジェクトドキュメント

詳しいファイルの役割は `README.md` を参照してください。
